# 📊 Projeto de Análise de Dados — SRAG

## 📌 Sobre o Projeto

Este projeto tem como objetivo realizar a análise exploratória de dados relacionados à Síndrome Respiratória Aguda Grave (SRAG), utilizando Python e Jupyter Notebook para tratamento, organização e visualização das informações.

O projeto busca identificar padrões, tendências e indicadores relevantes através de técnicas de limpeza de dados, análise estatística e visualização analítica.


## 🎯 Objetivos

- Realizar limpeza e tratamento dos dados
- Padronizar informações inconsistentes
- Explorar indicadores epidemiológicos
- Construir visualizações analíticas
- Identificar padrões temporais e demográficos
- Desenvolver análises voltadas à área da saúde e dados públicos


## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🧹 Etapa 1 — Limpeza e Tratamento dos Dados

Nesta etapa foram realizadas:

- Tratamento de valores ausentes
- Padronização de colunas
- Conversão de tipos de dados
- Remoção de inconsistências
- Organização do dataset para análise posterior


## 📊 Etapa 2 — Análise e Visualização dos Dados

As análises incluem:

- Distribuição de casos
- Análise temporal
- Indicadores demográficos
- Comparações estatísticas
- Construção de gráficos analíticos

> ⚠️ Projeto ainda em desenvolvimento. Novas análises e visualizações serão adicionadas futuramente.


## 📈 Exemplos de Análises

- Evolução temporal dos casos
